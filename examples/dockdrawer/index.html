Dock Drawer Animation Example

UX Animation Reference Prototype

Below is a list of animations perfomed in the dock drawer.

Sidebar Nav and Dock Drawer Animations
Sidebar Nav pinning or hovering is functional. Use hamburger menu to pin sidebar.
Dock Button - Click to open dock and view the drawer pinning.
Delete Item & Clear Items Animations
Drawer - Delete the "Endpoint Group" item to view remove animation and update the count.
Clear All - Presents confirmation to clear all and update the count.
Opening a Drawer Item Animations
Drawer - Open "L2 - Full Screen Dialog" to view opening animation.
L2 Dialog - Minimize and/or Close to view minimizing/closing animations.
L2 Dialog - When opened, reopen the dialog to see the drawer open just below the header bar.
